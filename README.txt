Developed a robust phishing detection system using Python, leveraging machine learning and domain analysis.
Implemented feature extraction techniques to identify malicious patterns in URLs, including length, special character counts, and domain age. 
Trained a RandomForestClassifier model achieving high accuracy in classifying URLs as benign or phishing. 
Integrated WHOIS data to analyze domain registration details, enhancing the system's ability to detect newly created or suspicious domains.
